O porque deste projeto

Este é um projeto criado com o propósito de aplicar na prática conceitos aprendidos em cursos como processamento de linguagem natural com spacy, utilização de APIS como fastAPI, programação em python para o backend,, aplicação de um banco de dados relacional, e a criação de uma página web em HTML, CSS e JavaScript . Além disso o projeto visa ser útil socialmente pois na conjectura atual muitas notícias falsas são espalhadas nas redes sociais com diferentes objetivos sejam eles difamatórios ou desmoralizador. Com este projeto tentamos combater ao menos um pouco o mar de desinformação que temos nas redes nos dias de hoje, de maneira prática e fácil de usar.

Metodologia

O nosso projeto utiliza de 2 metodologias, a primeira é a verificação da notícia em sites confiáveis. Através do título da notícia, procuramos em diferentes sites se a mesma já foi verificada e confirmada como verdadeira ou falsa. Caso isto ainda não tenha acontecido enviamos a notícia a nossa IA que irá verificar a probabilidade da notícia ser falsa com base em suas características de escrita e análise das informações e irar retornar ao usuário uma porcentagem da notícia ser falsa ou não. Por último caso a notícia possua uma alta porcentagem de ser falsa enviamos-a nosso banco de dados para que caso uma próxima pessoa nos envie a mesma notícia realizamos já teremos uma resposta mais rápida para a mesma.
